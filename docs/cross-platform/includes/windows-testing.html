测试说明
Xamarin.iOS
  • 开始进行测试的最简单方法是使用 Xcode 随附的 iPhone、iPad、Apple Watch 和 Apple TV 模拟器。使用 Visual Studio 进行调试时，可在连接的 Mac 上访问模拟器。
  • 若要使用设备进行测试，请按照这些说明操作（在连接的 Mac 计算机上执行大多数步骤）。
Xamarin.Android 按照这些说明来配置设备或使用仿真器：
  • Xamarin 安装程序包括“Google 仿真器管理器”，可用于配置 Google Android 仿真器进行测试。
  • Genymotion 和 Microsoft（通过 Visual Studio 2015）提供其他仿真器。
Xamarin.Forms 如上文所述，可将 Xamarin.Forms 应用部署到相关设备和仿真器。仅可通过连接的 Mac 硬件测试 iOS 应用；可直接在开发计算机上测试面向 Windows 8.1 或 UWP 的 Windows 平板电脑/桌面应用。
